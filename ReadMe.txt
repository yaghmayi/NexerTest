'Alpacinator' sample project is defined by Daniel Ekerot (daniel.ekerot@nexergroup.com)

Steps to run the project:
	1- Open 'NexerTest/Alpacinator.sln' file in Microsoft Visual Studio (2019 or newer)
	2- Run the project in Visual Studio (press Ctrl + F5)

In this project, we have used;
	- MVC Architecture + 3Layer programming
	- SqlLite (Memory database) + pure SQL for data access layer
	- UnitTests for Test assement
	- C# language and .net core framework in backend side
	- Angular and bootsatrab packages in frontend side
